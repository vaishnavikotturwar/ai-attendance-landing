# AI Attendance System - Landing Page

## 📌 Description
This is the frontend landing page for the AI Attendance System.

## 🚀 Features
- Clean UI design
- Responsive layout
- Connects with backend AI attendance system

## 🛠 Tech Stack
- HTML
- CSS
- Python (Flask)

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py
3. Open browser at:
   http://localhost:5000
